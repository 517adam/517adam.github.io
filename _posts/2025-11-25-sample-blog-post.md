This is a sample blog post to demonstrate how to add content to your site.

## How to Add a New Post

1.  Create a new file in the `_posts` directory.
2.  Name the file in the format `YYYY-MM-DD-title.md` (e.g., `2025-11-25-my-new-post.md`).
3.  Add the "Front Matter" at the top of the file (the text between the `---` lines).

### Example Front Matter

```yaml
---
title: "Your Post Title"
date: 2025-11-25
permalink: /posts/2025/11/your-post-title/
tags:
  - tag1
  - tag2
---
```

## Adding Content

You can use standard **Markdown** formatting for your posts.

*   Lists
*   [Links](https://google.com)
*   Images
*   Code blocks

Enjoy blogging!
